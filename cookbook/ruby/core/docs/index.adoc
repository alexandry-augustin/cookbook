== Installation

Install latest stable CRuby packages including RDoc, Psych, JSON, BigDecimal and IO/Console:

[,bash]
----
sudo dnf install ruby
----

[,bash]
----
sudo dnf install ruby-devel
----

== Usage

[,bash]
----
ruby filename.rb
----
