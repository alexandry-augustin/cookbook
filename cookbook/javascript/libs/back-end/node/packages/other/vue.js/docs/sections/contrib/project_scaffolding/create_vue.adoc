`create-vue`

[source,bash]
----
npm create vue@latest
or
npm init vue@3

cd prj_name
npm install
npm run dev
----

NOTE: Make sure your current working directory is the one where you intend to create a project.

----
npm install
----

[source,bash]
----
npm run dev
----

[source,bash]
----
npm run build
