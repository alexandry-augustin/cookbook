# Installation

```bash
sudo dnf install octave
```

# Usage

```bash
octave [options] file.m
```

## Flags

- `--persist filename.m`: 
- `--force-gui`: (since v3.8.2) Go to interactive mode after `--eval` or reading from a file named on the command line.
