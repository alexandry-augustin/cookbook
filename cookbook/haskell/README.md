# Glasgow Haskell Compiler (GHC)

## Installation

```bash
sudo apt-get install ghc
sudo dnf install ghc
```

## Usage

```
ghc
```

## REPL

```
ghci
```

- `quit`:
- `l filename`: (load `filename.hs`, provided that `filename.hs` is in same folder where `ghci` was invoked)
