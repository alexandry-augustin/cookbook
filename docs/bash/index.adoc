=== `printf`

....
printf format [arguments]
....

NOTE: `printf` does not automatically adds a new line character.

.Format Specifiers.
|===
| Character | Usage

| %s
| String

| %c
| Single character

| %d
| Integers

| %o
| Octal integers

| %x
| Hexadecimal integers

| %f
| Floating point

| %b
| String with backslash escape character

| %%
| Percent sign
|===
